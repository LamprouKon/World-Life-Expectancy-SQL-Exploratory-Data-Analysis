# 🌍 World Life Expectancy – SQL Exploratory Analysis

## 📌 Overview

This project performs structured exploratory data analysis (EDA) on a global life expectancy dataset using SQL.

The analysis focuses on identifying long-term health trends, measuring inequality between countries, and examining the relationship between economic and health indicators.


## 🔍 Key Areas of Analysis

- 📈 Life expectancy change over time (per country)
- 🌎 Global average life expectancy trend
- 💰 Relationship between GDP and life expectancy
- 🏦 High vs Low GDP segmentation
- 🌍 Developed vs Developing country comparison
- ⚖️ BMI and health outcome exploration
- 📊 Rolling mortality trends using window functions

## 📂 Project Structure

- `sql/eda_analysis.sql` → Structured SQL queries  
- `analysis_summary.md` → Insights & visual outputs
